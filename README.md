# kang.js <WIP>
A static site generator based on React, TypeScript, Vite and MDX.

### How to run this project
```bash
# 1. clone the project
git clone git@github.com:WWK563388548/kang.js.git

# 2. Install the npm globally
npm install -g

# 3. Build the project
pnpm start

# 4. Run the cli command
kang dev docs

```
